# Probabilistic Robotics
This project is aimed at providing solutions to the exercise problems of the textbook "Probabilistic Robotics".

## Requirment
- cmake
- opencv 3.3


## Build
     $ mkdir build
     $ cd build
     $ cmake ../
     $ make -j 8
